# id67_v2

*IDOBAO ID67 V2.0 (South-Facing RGB Version)*

* Keyboard Maintainer: [Logan Walls](https://github.com/LoganWalls)
* Hardware Supported: IDOBAO ID67 V2 (PCB: ID67H V2.0)
* Hardware Availability: [IDOBAO](https://www.idobao.net/collections/65-layout/products/idobao-id67-65-hot-swappable-mechanical-keyboard-kit-1), [Drop](https://drop.com/buy/idobao-id67-65-hot-swappable-mechanical-keyboard-kit)

Make example for this keyboard (after setting up your build environment):

    make id67_v2:default

Flashing example for this keyboard:

    make id67_v2:default:flash
